# Abstract Qiskit Algorithm on IBMQ (+ Demo UI)

> Deploys Python & Qiskit on an UbuntuVM and installs and runs Shor Qiskit algorithm on an IBMQ backend. A Demo UI is deployed as well.

## Properties

- `IBMQ_TOKEN`: Token of the IBMQ Experience Account.
- `IBMQ_BACKNED_NAME`: IBMQ backend, that should be used.
